Sobre

Code Filmes e Séries e um Agregador de conteúdo, através de codigos. Totalmente sem propaganda

Como Utilizar o Code Filmes e Séries?

Aqui vai um exemplo de como utilizar. Imagina que você deseja assistir a Série "Batwoman" pode usar o link "netflix3.shortcm.li/1/1"

Como Assim?

Sendo o Link Principal "netflix3.shortcm.li/" + 1/1" Onde o primeiro o numero será a Temporada e o segundo numero será o Epsodio na qual desejar assistir

Link Quebrado?

Utilize a "op2" logo após o link netflix3.shortcm.li/1/1/op2"

Observação

Nem sempre o link Principal será "netflix3.shortcm.li", cada Card contem a informção do link.
